Java整体概述

常用DOS命令：

  • dir：列出当前目录下的文件以及文件夹
  • md：创建目录
  • rd： 删除目录
  • cd： 进入指定目录
  • **cd..**：退回到上一级目录
  • cd: 退回到根目录
  • del： 删除文件
  • exit：退出dos命令行

Java语言的特点：

    1. 面向对象性：
      • 两个要素：类、对象
      • 三个特征：封装、继承、多态
    2. 健壮性：
        1. 强类型语言，在编译和运行时进行大量的类型检查，防止不匹配的数据类型发生
        1. 自动的垃圾回收机制，防止了内存分配的错误
        1. Java语言设计了异常处理机制
    3. 跨平台性：write once,run anywhere(原因：强大的JVM)

Java编程风格养成：

    1. 注释：Comment
      • 分类：
        • 单行注释：//
        • 多行注释：/* */
        • 文档注释：/** */
      • 作用：
        • 对所写程序进行解释说明，增强可读性。
        • 方便调式所写代码
    1. Java API文档：
      • API：application programming interface
      • API文档：提供类库使用教程
    1. 阿里巴巴Java开发文档(参考)

Java关键字的使用：

定义：专用字符串

特点：关键字全部小写

具体关键字：

保留关键字：goto 、const

标识符的使用：

  • 26个英文字母大小写、0~9数字、_或$组成
  • 数字不可以开头
  • 不可以使用关键字和保留字
  • Java中严格区分大小写、但长度无限制
  • 标识符不能包含空格

注意点：为了提高阅读性，尽量“见名知意”。
